$variables

Before 

$primary-font: Helvetica, Arial, sans-serif;
$secondary-font: Monoco, sans-serif;
$primary-color: #F00;

h1,h2,h3 {
  font-family: $primary-font;
  color: $primary-color;
}

article {
  font-family: $secondary-font;
}
After 

h1,h2,h3 {
  font-family: Helvetica, Arial, sans-serif;
  color: #333;
}

article {
  font-family: Monoco, sans-serif;
}

Importing

headlines.scss 

h1 {
  font-size: 36px;
}
h2 {
  font-size: 24px;
}
footer.scss 

.footer {
  background: #333;
  color: white;
}
main.scss 

@import "headlines";
@import "footer";
main.css 

h1 {
  font-size: 36px;
}
h2 {
  font-size: 24px;
}
.footer {
  background: #333;
  color: white;
}

Nesting

Before 

nav {
  ul {
    margin: 0;
    list-style: none;
  }

  li {
    display: inline-block;

    a {
      display: block;
      padding: 6px 12px;

      &.purple { color: purple; }
    }
  }
}
After 

nav ul {
  margin: 0;
  list-style: none;
}

nav li {
    display: inline-block;
}

nav li a {
  display: block;
  padding: 6px 12px;
}

nav li a.purple {
  color: purple;
}

  • What are Best Practices?

  • Can't hire for it

  • Very secure
  • Best Practices & Standards
  • Public documentation
  • Easy to find developers

ORM - Object Relation Mapping

  • Simplifies Database Communication
  • DB Tables represented as PHP Objects
  • Automatically secure
Without ORM 

$query = "SELECT * FROM flights WHERE active='1' ORDER BY name DESC";

if ($result = $mysqli->query($query)) {

  while ($row = $result->fetch_assoc()) {
    echo $row['destination'];
  }

}
With ORM 

$flights = Flights::where('active',1)->orderBy('name','desc')->get();

foreach ($flights as $flight) {
  echo $flight->destination;
}
